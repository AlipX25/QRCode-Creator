# QRCode-Creator
Website sederhana yang digunakan untuk membuat link QRCode

# Setup
- `git clone https://github.com/RaihanNih/QRCode-Creator.git`
- `cd QRCode-Creator`
- `composer install`
- `php -S localhost:8000`
